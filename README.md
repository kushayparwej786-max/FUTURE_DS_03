# Marketing Funnel & Conversion Performance Analysis

## Project Overview
This project analyzes marketing funnel data to understand how users move from visitors to leads and finally to customers.  
The goal is to identify conversion drop-offs and discover opportunities to improve marketing performance.

## Objectives
- Analyze customer conversion patterns
- Identify key factors affecting conversions
- Understand customer demographics and behavior
- Provide insights to improve marketing campaigns

## Dataset
The dataset used in this project is the **Bank Marketing Dataset**, which contains information about customers contacted during marketing campaigns.

Main features include:
- Age
- Job
- Marital status
- Education
- Account balance
- Loan status
- Campaign information
- Deposit subscription (Yes/No)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Analysis Performed
The following analyses were performed:

1. Customer Conversion Distribution  
2. Age Distribution of Customers  
3. Job Category vs Conversion  
4. Account Balance vs Conversion  

These analyses help understand which customer segments are more likely to subscribe to a deposit.

## Key Insights
- Most customers did not subscribe to the deposit.
- Certain job categories show higher conversion rates.
- Customers with higher balances tend to convert more.
- Age distribution can help identify target marketing segments.
